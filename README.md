# Simple-Calculator
My fourth Python project

## Features
- Prompt the user to enter two numbers and store them in variables
- Handle cases where the input is not a number (e.g., text or other characters)
- Prompt the user to choose one of the following operators (+, -, *, /)
- Handle cases where the entered operator does not match the options provided (e.g., text or other characters)
- Display numbers without a decimal point if the two input numbers and the result are not decimals
- Handle division by zero

## Technologies
- Python 3
